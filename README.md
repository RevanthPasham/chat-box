## 🚀 Setup Instructions

->Install Dependencies
```
pip install Flask google-generativeai
```
->Add Your Gemini API Key
->Open app.py and replace:
```
GOOGLE_API_KEY = "YOUR_GOOGLE_API_KEY"
```
With your actual API key.

->Run the App
```
python app.py
```
